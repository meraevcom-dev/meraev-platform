# MeraEV

MeraEV is India's EV Intelligence Platform helping consumers compare electric vehicles, discover charging stations, calculate ownership costs, and make smarter EV buying decisions.

## Features

- EV Comparisons
- Charging Station Finder
- EV Specifications
- EMI Calculator
- Battery Insights
- Dealer Discovery
